# NeatChests

A Minecraft plugin/mod designed to bring smarter, cleaner, and more efficient chest management to your world. 

## Features
* 📦 **Smart Sorting:** Keep your chests organized instantly without the hassle.
* ⚡ **Lightweight & Fast:** Highly optimized to ensure zero performance impact on your server.
* ⚙️ **Config File:** *Coming soon!* (Currently running on optimal default settings).

## Compatibility
> ⚠️ **Important:** NeatChests currently supports Minecraft version **26.2** and above. Older versions are not officially supported.

## Installation

1. Download the latest release `.jar` file.
2. Drop the file into your server's `plugins` (or `mods`) folder.
3. Restart your server.
4. Enjoy your neatly organized chests!

## How to Use

1. Open any chest.
2. **Double-click** on an empty slot.
3. *Voilà!* Your chest is now sorted.
